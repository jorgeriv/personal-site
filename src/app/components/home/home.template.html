Productivity is key

Deliver, deliver, deliver... but don't forget quality

I'm a firm believer in automation, as a developper, I love to make machines perform repetitive time-consuming and tedious task for me. An automated development environment is a good development environment. Whitout automation, the act of programming is tedious and error prone. Once you automate all your tedious tasks, you are a happy and productive developper. There is no reason for a human been to run code linting process, tests, building tasks and code deployment manually. Automation equials better quality products developed in less time.
